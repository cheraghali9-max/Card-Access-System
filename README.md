 Card Access System (A simple C project )

 Description of the project 
This project is a simple card access system written in C.  
It simulates a door access system where users can add, remove, and check cards.

 Features of the project
- Open door with valid card
- List all cards in the system
- Add and remove cards (dynamic memory with malloc/realloc)
- Prevent duplicate cards
- Check fake cards

Structure of the project
- main.c → starts the program
- menu.c → all logic and menu handling
- menu.h → structure and function declarations
- saveinput.c / saveinput.h → safe user input handling

 How to Compile?
Using  GCC

What I learn from this first C project?
- Dynamic memory (malloc, realloc, free)
- Arrays and structures in C
- Menu-based program design
- Input validation without scanf
